## GitHub API  获取仓库目录

#####  基地址：`https://api.github.com`

### 默认前1k个文件

```
GET /repos/{owner}/{repo}/contents/{path}

owner：所有者
repo：仓库名字
path：路劲
```

### 全部文件

```
GET /repos/{owner}/{repo}/git/trees/{tree_sha}

owner：所有者
repo：仓库名字
tree_sha：目录的sha值,可以通过上面的接口拿到
```



## jsDelivr CDN加载仓库文件
